1. INTRODUCTION

This Privacy Policy describes how the TV QR Code application (hereinafter referred to as "Application", "we", "us", or "our") collects, uses, stores, and protects user information.

By using this Application, you agree to the terms of this Privacy Policy. If you do not agree with any of the terms, please do not use the Application.

2. GENERAL INFORMATION ABOUT THE APPLICATION

TV QR Code is an Android TV application designed to display QR codes and promotional materials (images and videos) on television screens. The application operates in kiosk mode and is intended for commercial use.

3. INFORMATION WE COLLECT

3.1. Device Identifiers

The Application collects and uses the following identifiers:

  • Android Device ID — a unique Android device identifier used to identify your device when connecting to the server
  • This identifier is used solely to ensure the Application's functionality and cannot be used to identify a specific user

3.2. Connection Information

The Application collects the following network connection information:

  • Internet connection status
  • Time of last successful connection to the server (stored locally on the device)
  • Network status information to ensure stable Application operation

3.3. Technical Information

The Application may collect the following technical information:

  • Application version
  • Error and crash information (to improve Application performance)
  • Authorization tokens (stored locally on the device to ensure connection security)

3.4. Data We Do NOT Collect

The Application does NOT collect the following information:

  • Personal user data (name, email address, phone number, etc.)
  • Device location
  • User contacts, photos, or other files
  • Information about other applications on the device
  • Payment or financial transaction information

4. USE OF COLLECTED INFORMATION

The collected information is used solely for the following purposes:

4.1. Ensuring Functionality

  • Device identification when connecting to the server
  • Displaying QR codes and promotional materials associated with your device
  • Ensuring stable connection to the server
  • Real-time content updates

4.2. Improving Application Performance

  • Monitoring connection status
  • Detecting and fixing errors
  • Performance optimization

4.3. Security

  • Device authentication on the server
  • Protection against unauthorized access
  • Ensuring data integrity

5. DATA STORAGE

5.1. Local Storage

The following data is stored locally on your device:

  • Authorization tokens
  • Time of last successful connection
  • Application settings

This data is stored in the application's secure storage and is not shared with third parties without your consent.

5.2. Server Storage

The following data may be stored on the server:

  • Android Device ID (for device identification)
  • Application version
  • Device connection status

Data is stored on servers located at: https://trade.aplit-soft.com

5.3. Retention Period

  • Data is stored for the period necessary to ensure the Application's functionality
  • When the Application is deleted, all locally stored data is deleted along with the Application
  • Server data may be stored for the period necessary to provide services

6. SHARING DATA WITH THIRD PARTIES

6.1. Server Services

The Application connects to the server at https://trade.aplit-soft.com:4343 for:

  • Receiving QR codes
  • Downloading promotional materials (images and videos)
  • Downloading company logos
  • Device authentication

Data is transmitted only to the specified server and is used solely to ensure the Application's functionality.

6.2. Third-Party Services

The Application uses the following third-party libraries:

  • OkHttp — for network requests
  • Glide — for image loading and caching
  • ExoPlayer — for video playback
  • ZXing — for QR code generation

These libraries may have their own privacy policies, which you can review on their official websites.

6.3. Disclosure of Information

We do not sell, rent, or share your data with third parties, except in cases:

  • When necessary to ensure the Application's functionality
  • When required by law or by request of government authorities
  • To protect our rights and security

7. APPLICATION PERMISSIONS

The Application requests the following permissions:

7.1. INTERNET

Required to connect to the server and download content.

7.2. ACCESS_NETWORK_STATE

Required to check internet connection status and ensure stable operation.

7.3. RECEIVE_BOOT_COMPLETED

Required for automatic Application launch after device reboot (kiosk mode).

7.4. WAKE_LOCK

Required to keep the screen on during Application operation.

7.5. SYSTEM_ALERT_WINDOW

Required for the Application to operate in kiosk mode.

7.6. SCHEDULE_EXACT_ALARM

Required for scheduling content updates.

7.7. REQUEST_IGNORE_BATTERY_OPTIMIZATIONS

Required to ensure stable Application operation in the background.

All permissions are used solely to ensure the Application's functionality and are not used to collect additional information.

We take reasonable measures to protect your data:

However, despite the measures taken, no method of data transmission over the internet is absolutely secure. We cannot guarantee absolute data security.
